This tutorial will teach you how to make AJAX call using both plain JavaScript as well as jQuery

What we will learn during ths tutorial?

  1. What AJAX, HTTP, XML and JSON are
  2. Read files using AJAX
  3. Read JSON using AJAX
  4. Call REST API using AJAX
  5. Use AJAX in jQuery

How to start?

1. Prepare the environment

You can check this tutorial on how to setup environment.

1.1. Text editor

Download and install Visual Studio Code

1.2 Browser

Download and install one of the following browser:

  • Google Chrome
  • Firefox
  • Opera

1.3 Lesson files

Download or clone lesson files into your computer.

You will see the following files

        tutorial/ 
            1-start-here.html
            2-ajax.html
            3-ajax-text-file.html
            4-ajax-json.html
            5-ajax-api.html
            6-ajax-jquery.html
        final/
            ajax-text-file.html
            ajax-json.html
            ajax-api.html

2. Start learning

Navigate to tutorial folder and follow lesson files in given order:

  1. 1-start-here.html (current file)
  2. 2-ajax.html
  3. 3-ajax-text-file.html
  4. 4-ajax-json.html
  5. 5-ajax-api.html
  6. 6-ajax-jquery.html
