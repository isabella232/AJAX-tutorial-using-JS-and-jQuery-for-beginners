AJAX calls using jQuery

Let's use our previous code from ajax-api.html file. In order to use jQuery we obviously have to load it first


    <script src="https://code.jquery.com/jquery-3.5.1.min.js" integrity="sha256-9/aliU8dGd2tb6OSsuzixeV4y/faTqgFtohetphbbj0=" crossorigin="anonymous"></script>

Now we can rewrite our JS code with jQ, first listener:



    $(function() {
        $('button').on('click', makeCall)

    })

Now makeCall function:


    function makeCall(){
        $.ajax({
            type: 'GET',
            url:  'http://api.openweathermap.org/data/2.5/weather?q=London&appid=f6a5f1615eaaf79b77fde5de66d2bf4a&&units=metric',
            success: function(response) {
                console.log(response);
            },
            error: function(e){
                console.log(e);
            }
        })

As you can see we it's very easy. Using $.ajax() we have to specify a post method (GET), and endpoint with parameters, and decide what to do when our call is successful and what if not.
